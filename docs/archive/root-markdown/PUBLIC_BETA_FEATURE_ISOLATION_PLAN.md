# Public Beta Feature Isolation Plan

Date: 2026-05-20

## Feature Flags

- Marketplace enabled.
- Uploads enabled.
- Messaging enabled.
- Public registration enabled.
- Admin moderation enabled.

## Admin-Only

- Marketplace moderation.
- Report review.
- User restrictions.
- Operational dashboards.

## Hidden/Beta-Only

- Experimental marketplace features.
- Bulk upload flows.
- Public beta registration if invite-only mode is active.

## Kill Switches

- Disable uploads.
- Disable messaging.
- Disable listing publication.
- Disable registration.
