# Production Rollout Phases

Date: 2026-05-20

## Phase 1 Internal Alpha

- Team-only access.
- Validate auth, lab, breeder, upload, marketplace core flows.

## Phase 2 Invite Beta

- Small trusted breeder/lab group.
- High-touch support.

## Phase 3 Breeder Beta

- Expand breeder workflows.
- Keep marketplace limited.

## Phase 4 Marketplace Beta

- Enable listing/media/messaging with moderation ready.

## Phase 5 Public Launch

- Public registration.
- Monitoring and support staffed.
- Legal/compliance approved.

## Rollback Triggers

- Auth/session incident.
- Data integrity issue.
- Marketplace abuse without mitigation.
- Critical workflow regression.
