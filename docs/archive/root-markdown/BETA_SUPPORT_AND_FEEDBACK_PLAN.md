# Beta Support And Feedback Plan

Date: 2026-05-20

## Channels

- One primary support inbox or ticket system.
- One structured feedback form.
- One internal triage board.

## Triage Levels

- Critical: login unavailable, data loss, payment/security issue.
- High: core lab/breeder workflow blocked.
- Medium: workaround exists.
- Low: copy, layout, minor UX.

## Feedback Fields

- User role.
- Browser/device.
- Workflow.
- Expected behavior.
- Actual behavior.
- Screenshot or recording if available.
