# Beta Feature Flag Plan

Date: 2026-05-20

## Recommended Flags

- Marketplace enabled.
- Marketplace uploads enabled.
- Marketplace messaging enabled.
- Marketplace report/block enabled.
- Lab order creation enabled.
- Certificate download enabled.
- Public registration enabled.

## Kill Switches

- Disable uploads.
- Disable marketplace messaging.
- Disable public listing publication.
- Disable public beta registration.

## Rule

Flags should default conservatively for beta and be documented per deployment environment.
